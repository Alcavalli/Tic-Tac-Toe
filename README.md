# Tic-Tac-Toe

Recreated the classic Tic-Tac-Toe game in C++ with the library SFML.
Different gamemode and difficulty to choose from.


## Dependencies

- CMake 3.10+
- SFML 3


## How to compile

```bash
mkdir build
cd build
cmake ..
cmake --build .
./main
```

or, in VS Code, just Ctrl + Shift + P > CMake: build


## How to play

Select the settings you prefer the most and click on the cell you want to put your symbol on.


## Folder structure

```
src/        # source files (.cpp)
include/    # header files (.hpp)
assets/     # font and graphics
```
